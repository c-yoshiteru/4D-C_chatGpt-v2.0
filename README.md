AIとの集団コウサク実践フレームワーク

---

## 概要（Overview）

**4D-C** は、人間とAI、そして複数人による協創（集団コウサク）を  
「意志（C）」と「観測点（マリ）」の安定性という観点から整理・実践するための理論・プロトコルです。

本リポジトリは、  
- 思想・理論（Concept）
- 実践プロトコル（Practice）
- 可視化（Diagram）
- 応用（Application）

を **Markdownベースで段階的に整理** したものです。

コードが分からなくても読める構成を意識しています。

---

## このリポジトリで扱うこと

- 4D-C理論 v2.0 の基本構造
- 「オラー」「謙虚さ」「不安」という感情ベクトルの扱い
- コヒーレント／インコヒーレント状態の分岐
- AIとの **集団コウサク（Collective Co-creation）** の実践指針
- GitHub上での知識構造化の実例
