
# A1. 4D-C v2.0 概要

4D-C（Four-Dimensional Coherence）は  
**「オラー（軸）」と「謙虚さ（柔軟性）」の同時成立**を中心に据えた  
意識状態のモデルである。

v2.0 では、次の 4 つを主要指標として定義する：

- **Stability（有限化）** … 「オラァー」の強度  
- **Inversion（反転）** … 「謙虚さ（柔軟性）」の強度  
- **C値（Coherence Value）** … 真実性のシグナル  
- **Compression（圧縮率）** … 意志とノイズの乖離度

さらに、C値に応じて 5 つの MariStage を分類する：

1. CHAOS  
2. INVERT  
3. ENTRAIN（予備）  
4. SYNC  
5. UNITY  

これらの値は `StateAnalyzer` クラスによって算出される。

本フォルダ「A_コンセプト」では、  
4D-C v2.0 の基本思想と、各指標の意味をまとめる。
